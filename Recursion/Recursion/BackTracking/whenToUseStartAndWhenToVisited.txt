start mtlb ek variable jo sirf aage move krne me help krta hai.

Suppose 

func(int nums[]){
    for(int i = 0; i < nums.length; i++){
        func(nums); -> recursion call
    }
}

Upar example me jab bhi recursion call hoga, i = 0 se start hoga. Isko tackle krne ke lie hum start
variable introduce krte hai jese

func(int nums[], int start){
    for(int i = start; i < nums.length; i++){
        func(nums, i+1); -> recursion call
    }
}

start increment hamesha i+1 se hoga na ki start+1 se, kyuki for loop ke ander recursion call kar
ho to update i hoga na ki start. Or ye janna hai ki abhi tak kaha tak traversae kiya hai
to i bataega, start to sirf ye bataega ki current recusion branch start kaha se hui hai.
