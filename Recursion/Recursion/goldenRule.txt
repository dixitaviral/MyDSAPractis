## 2. Har recursion me 3 cheezein fix hoti hain

### (1) Base case

Jahan function **ruk jaata hai**.
Nahi hoga â†’ **infinite recursion / stack overflow**.

---

### (2) Smaller call

Problem ko **size me chhota** karna zaroori.

Examples:

* `n â†’ n-1`
* `n â†’ n/2`
* `n â†’ n/10`
* tree â†’ `left` & `right`

Agar size **chhota nahi ho raha** â†’ recursion galat.

---

### (3) Combine step

Recursive call ka **result use** karke final answer banana.

Examples:

* Sum â†’ `n + f(n-1)`
* Even sum â†’ `2n + f(n-1)`
* Power of two â†’ **sirf result return**
* Tree count â†’ `left + right + 1`
* Tree height â†’ `max(left, right) + 1`

## 4. Recursion ke shapes

### Linear recursion


### Two-way recursion
